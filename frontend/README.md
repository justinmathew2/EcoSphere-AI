# EcoSphere-AI Frontend

This is the React frontend for the EcoSphere-AI project.

## Problem Statement
AI-powered Carbon Footprint Awareness Platform using Google Gemini to help users track, understand, and reduce their environmental impact through personalized sustainability insights.

## Overview
The frontend is built with React, Vite, and TypeScript. It interfaces with the FastAPI backend and provides an interactive UI for users to:
- **Track**: Calculate their carbon footprint dynamically using the Carbon Calculator.
- **Understand**: Use AI to analyze electricity bills and scan products for sustainability ratings.
- **Reduce**: Receive personalized weekly action plans and chat with the AI Sustainability Coach.

## Setup
```bash
npm install
npm run dev
```
